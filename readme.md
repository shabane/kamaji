### Automated VPN Configuration Aggregator & Tester

**Kamaji** is a powerful tool designed to scrape, test, and aggregate VPN configurations (**Shadowsocks, Vmess, Vless, and Trojan**) from various Telegram channels.

Many public configurations expire quickly or become blocked due to high traffic. Kamaji solves this by automatically collecting these links and performing connectivity testsâ€”specifically verifying connections via a server located in **Iran**â€”to ensure reliability. The result is a set of clean, constantly updated subscription files ready for your VPN client.
